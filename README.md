# Droplets

A weather application that displays current weather conditions for any city using the Visual Crossing API.

## Features

- Search weather by city name.
- Displays temperature, weather conditions, feels like, humidity, precipitation probability, wind speed for the next 15 days.
- Error handling for invalid cities and failed requests.
- Celcius and Fahrenheit toggle.

## Built with

- Javascript (ES6+)
- Webpack
- Visual Crossing API
- HTML/CSS

## Dev Tools

- ESLint - code linting
- Prettier - code formatting

## What I learned

- Working with external APIs and handling async operations
- Managing submission state to prevent duplicate API requests
- Error handling for different failures (network errors, missing DOM elements)
- Learning to utilize CSS Grid and Flexbox more efficiently.

## Getting Stared

1. Clone the repository
   `git clone https://github.com/SuspiciousApplePie/droplets.git`
2. Install dependencies
   `npm install`
3. Run the development server
   `npm run start`
